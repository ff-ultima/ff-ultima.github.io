Setup

FF Ultima doesn't depend on autoconfig, but it can be set up and used in the same way as default firefox with the help of the .autoconfig/ folder. The .autoconfig/ folder sources it's setup files from https://github.com/AlexVallat/firefox-scripts?tab=readme-ov-file, and these files include 1 minor adjustment that allows users to organize their custom scripts in chrome/scripts/. Here's how to set it up:

In the .autoconfig/ folderâ€‹

  1. Move everything in .autoconfig/chrome/ into the original chrome folder.
  2. Move everything in .autoconfig/firefox/ into your firefox installation folder.
  3. In Firefox, navigate to about:support and click on Clear Startup Cache...

Once Firefox restarts, you'll notice some new buttons and features appear in Firefox. These are all features sourced from the chrome/scripts folder.
